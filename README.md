# How to run this project
clone this repo Install NodeJS cd into the repo and run:

`npm install`

`npm install webpack-cli -g`

`npm install webpack -g`

`npm install concurrently -g`

`npm install nodemon -g`

`npm run start`

`open http://localhost:3000/`
